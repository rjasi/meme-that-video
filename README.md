# meme-that-video
A tool to download all the frames of some video so you can meme that specific moment

## Usage

`python main.py --url "https://www.youtube.com/watch?v=DpF44k9aBUc" --output_dir "./temp" --start_timestamp 5 --stop_timestamp 6`

## Tested sites
1. youtube.com
